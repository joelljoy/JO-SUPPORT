# ☕ Support Me - UPI Donation Page

A modern Buy Me a Coffee style donation page** that allows supporters to send payments through UPI apps like Google Pay, PhonePe and Paytm.

This project provides a clean and attractive interface where visitors can support the creator by sending small donations.

---

## ✨ Features

- ☕ Predefined donation buttons (₹10, ₹30, ₹50, ₹100)
- 💬 Supporter message input
- 📱 Direct UPI payment redirection
- 🔳 UPI QR code for desktop users
- 🎨 Modern glassmorphism UI
- ✨ Animated UI effects
- 📊 Supporter wall display
- 📱 Mobile friendly responsive design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Tailwind CSS

---

## 💳 Payment Methods Supported

Users can pay using:

- Google Pay
- PhonePe
- Paytm
- Any UPI supported banking app

The payment is processed using a **UPI deep link** that redirects users directly to their payment app.
